# iframe-promise
Promise can be used for iframe cross domain communication

# Installation

```bash
npm instal --save-dev @emojiiii/iframe-promise
```

```bash
yarn add @emojiiii/iframe-promise
```

```bash
pnpm add @emojiiii/iframe-promise
```
